# Quantum Digital Circuits using PennyLane

## Overview

This project demonstrates the implementation of quantum digital circuits using the PennyLane quantum computing framework.

The project contains:

- Quantum 2:1 Multiplexer
- Quantum Multiplier
- Classical Verification
- Power Estimation
- Performance Analysis
- Data Visualization

## Features

- Quantum Multiplexer
- Quantum Multiplier
- Truth Table Generation
- Classical Verification
- Power Estimation
- Graph Generation

## Technologies Used

- Python
- PennyLane
- NumPy
- Matplotlib

## Installation

Clone the repository

```bash
git clone https://github.com/varsha-valavan/Quantum-Digital-Circuits-with-PennyLane.git
```

Install packages

```bash
pip install -r requirements.txt
```

Run

```bash
python src/quantum_mux.py
```

or

```bash
python src/quantum_multiplier.py
```
